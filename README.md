# Donteatthecashier – Browser Visual Novel Game

Donteatthecashier is a dark comedy monster visual novel where you work the night shift at a gas station serving mysterious monster customers.

Every customer you meet can change your story. Some are friendly, some are dangerous, and some unlock unexpected narrative paths.

## 🧠 About the Game

Donteatthecashier is a story-driven visual novel combining:

- Dark comedy storytelling  
- Monster romance elements  
- Horror-themed atmosphere  
- Branching dialogue choices  
- Multiple endings and replayable routes  

The game focuses on narrative depth, character interaction, and surreal storytelling inside a strange gas station at night.
